# 🌤 SkyCast — Modern Weather Dashboard

SkyCast is a modern animated weather dashboard that provides real-time weather insights with an interactive and visually rich user interface.

The application integrates live weather APIs to display current weather conditions, air quality data, hourly forecasts, and a 7-day forecast. It also includes a dynamic sun-path visualization that shows daylight progress between sunrise and sunset.

## ✨ Features

- 🌦 Real-time weather data using OpenWeather API
- 📊 Hourly temperature forecast
- 📅 7-day weather forecast
- 🌫 Air Quality Index (AQI) monitoring
- 🌅 Sun path visualization for sunrise & sunset tracking
- 🗺 Interactive weather radar map
- 🌙 Dynamic day / night UI
- 📱 Fully responsive design
- 🎨 Smooth animations and modern UI

---

## 🛠 Tech Stack

- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **Animations:** Framer Motion  
- **API:** OpenWeather API  
- **Maps:** Leaflet.js  
- **Build Tool:** Vite  
- **Deployment:** Vercel

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/harshitgoel006/SkyCast.git

Navigate into the project

cd SkyCast

Install dependencies

npm install

Run the development server

npm run dev

🔑 Environment Variables

Create a .env file and add your OpenWeather API key

VITE_WEATHER_API_KEY=your_api_key_here

You can get a free API key from:

https://openweathermap.org/api
